# Kickoff Call Checklist

**When:** Day of deposit clearance. Within 48 hours.
**Duration:** 30 minutes.
**Format:** Zoom (record it, save to `clients/<slug>/03-kickoff-notes.md`).

## Agenda

### 1. Welcome (3 min)
- Re-confirm: what they want, deadline, their one-line goal.
- "Before we dig in — anything changed since we talked last?"

### 2. Walk through the plan (10 min)
- Open the signed scope/proposal side-by-side.
- Confirm each deliverable is still accurate.
- Share the delivery timeline with actual dates. Lock check-in cadence.

### 3. Access + assets checklist (10 min)
Go through `ops/credentials-policy.md` with them live. Tell them exactly what you need and the secure way to send it.

Needed for most projects:
- Domain registrar login (or new domain purchase)
- Hosting access (or permission to set up new)
- Google account for Analytics + Search Console (or permission to create)
- Existing logo files + brand assets
- For chatbot: knowledge base answers
- For blog: topic list + keywords
- For commercials: brand visuals + any reference reels

### 4. Communication rules (5 min)
- Primary channel (email / Slack / text)
- Response time expectations (yours + theirs)
- Who signs off on decisions (if there are multiple stakeholders, get ONE)
- Emergency contact (yours: 702-907-5071)

### 5. Close (2 min)
- Confirm next checkpoint date.
- Send follow-up email within 1 hour with: notes, next steps, asset request list.

## After the call — same day

- [ ] Save recording to client folder
- [ ] Write summary to `clients/<slug>/03-kickoff-notes.md`
- [ ] Send follow-up email with:
    - Bullet summary of what was agreed
    - Asset request list with deadline (3 business days)
    - Next check-in date
    - "If anything in this summary is wrong, reply within 24 hours"
- [ ] Start building.
