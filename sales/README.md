# Sales

Everything used BEFORE a client signs. In order of the sales cycle:

1. **Discovery call** — `discovery-call-script.md`
2. **Objection handling** (during the call) — `objection-handlers.md`
3. **Proposal** — `proposal-template.md` (send within 24hrs of the call)
4. **Pricing reference** — `pricing-sheet.md`
5. **Intake form** — `intake-form.md` (send once they say "let's do it")
6. **Contract** — `contract-template.md` (send with first invoice)

Once they sign → move to `ops/onboarding-runbook.md`.
